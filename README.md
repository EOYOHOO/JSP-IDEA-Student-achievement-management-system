# JSP-IDEA-Student-achievement-management-system
web课程设计

本项目为IDEA项目，项目名为学生成绩管理系统。

本项目具有两个身份 学生和管理员。

管理员账户默认初始账号密码为 admin 123456。

学生账户需要在管理员账户中新建学生信息才能进行登录。
